# 上海博物馆AI导览 - 卢浮宫风格前端设计方案

## 1. 设计理念

### 1.1 灵感来源
- **卢浮宫官网**：优雅的色彩搭配，强调视觉冲击力
- **大英博物馆**：专业的信息架构和用户体验
- **故宫博物院**：中国传统元素的现代演绎

### 1.2 核心设计原则
- **奢华感**：金色、深蓝色的配色方案
- **历史感**：优雅的字体和布局
- **现代感**：流畅的动画和交互
- **专业感**：清晰的信息架构

## 2. 色彩系统

### 2.1 主色彩
```css
:root {
  --color-primary: #D4AF37; /* 金色 */
  --color-secondary: #1A237E; /* 深蓝色 */
  --color-accent: #C62828; /* 红色点缀 */
  --color-dark: #263238; /* 深灰黑 */
  --color-light: #F5F5F5; /* 浅灰 */
  --color-white: #FFFFFF;
  --color-text: #333333;
  --color-text-light: #666666;
}
```

### 2.2 应用场景
- **主色调**：金色用于强调和按钮
- **背景**：深蓝色渐变背景
- **文字**：深灰色和白色
- **边框**：金色边框用于突出元素

## 3. 布局设计

### 3.1 展品列表页
- **背景**：全屏深蓝色到金色的渐变
- **标题**：居中大标题，金色文字
- **展品卡片**：悬停效果，金色边框，精致阴影
- **布局**：响应式网格布局

### 3.2 聊天界面
- **顶部**：大幅展品图片，半透明导航栏
- **导航**：金色返回按钮，展品信息
- **控制栏**：深度控制和TTS控制
- **聊天区域**：优雅的对话气泡
- **输入区**：金色发送按钮

## 4. 交互设计

### 4.1 动画效果
- **页面切换**：淡入淡出过渡
- **元素加载**：优雅的加载动画
- **悬停效果**：卡片放大，按钮变色
- **滚动效果**：平滑滚动

### 4.2 微交互
- **按钮反馈**：点击和悬停状态
- **深度切换**：金色渐变按钮
- **TTS控制**：精致的播放/暂停按钮
- **通知**：优雅的Toast提示

## 5. 字体系统

### 5.1 字体选择
- **标题**：Playfair Display（衬线字体）
- **正文**：Inter（无衬线字体）
- **按钮**：Roboto（清晰易读）

### 5.2 字体大小
- **大标题**：3rem
- **小标题**：1.5rem
- **正文**：1rem
- **小文本**：0.875rem

## 6. 技术实现

### 6.1 技术栈
- **前端框架**：React + TypeScript
- **样式**：Tailwind CSS（自定义配置）
- **动画**：Framer Motion
- **图标**：React Icons

### 6.2 组件结构
```
frontend/
├── src/
│   ├── components/
│   │   ├── ExhibitCard.tsx        # 展品卡片
│   │   ├── ChatMessage.tsx        # 聊天消息
│   │   ├── DepthControl.tsx       # 深度控制
│   │   ├── ToastNotification.tsx   # 通知
│   │   └── LoadingSpinner.tsx     # 加载动画
│   ├── styles/
│   │   └── globals.css            # 全局样式
│   ├── App.tsx
│   └── main.tsx
```

## 7. 响应式设计

### 7.1 断点设置
- **移动端**：< 768px
- **平板**：768px - 1024px
- **桌面**：> 1024px

### 7.2 布局适配
- **移动端**：单列布局，优化触摸交互
- **平板**：双列布局，适当放大元素
- **桌面**：三列布局，更大的视觉冲击力

## 8. 性能优化
